# Ziqi (Charlie) Liu

Static academic homepage for GitHub Pages.

## Local Preview

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deploy

Use this repository as `sunbus100.github.io`, push to the `main` branch, and enable GitHub Pages from the branch root.

## CV

Place the CV at:

```text
assets/Ziqi_Liu_CV.pdf
```

The homepage only shows the CV link when that file is available.
